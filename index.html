Ayurvedic Traceability

A blockchain-based traceability system for Ayurvedic herbs — farmer portal, geo-tagging, QR per batch, admin dashboard & customer view.

Farmer Portal Admin Dashboard Customer View

Project summary

This demo page shows the frontend only. Farmers & collectors add herb details with geo-tagging; each batch has a QR code that consumers scan to view the journey from collection → processing → final product.

  • Farmer portal (add herb batch, geo-tag)
  • Blockchain ledger (immutable entries — simulated)
  • QR generation per batch
  • Admin dashboard (review & verify)
  • Customer view (scan QR → trace history)
